# People Management — FE Interview Task

This repo contains a take-home task for a Frontend Engineer position.

## Getting started

### 1. Clone the repo

```bash
git clone <repo-url>
cd nexxen-fe-interview
```

### 2. Start the backend

```bash
cd backend
npm install
npm start
# API running on http://localhost:3000
```

### 3. Start the frontend

```bash
cd frontend
npm install
npm run dev
# App running on http://localhost:5173
```

### 4. Open the design mockup

Open `design/mockup.html` in your browser — it shows all 3 screens you need to implement (click the screen buttons at the top to navigate between them).
